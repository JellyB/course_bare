教师介绍
华图教育研究员，公务员实战专家，行测科目业内名师，在公考领域有其独树一帜的授课风格。先后参与编写《华图名师成”功”系列教材》《2012国考行政职业能力测验教材》《2012军转干行政职业能力测验教材》《微魔块系列之数量资料36计》等一系列业内经典教材。 在讲台上，刘老师总是激情四射，总能将复杂的数学题目讲的娓娓动听，其幽默风趣的个性特点使得课堂上总是充满了欢声笑语。 刘老师坚信优秀的老师一定要了解学生、灵活运用教学方法；一定要学会倾听，倾听自己的课堂。在课堂上，学会了真正倾听学员的心声，从学员们真正的学习需要出发，组织教学。
